## CRM Application – Spring Boot

This is a simple CRM (Customer Relationship Management) application
developed while learning backend development using Spring Boot.

The project focuses on understanding how backend applications manage
users, roles, and basic business data through REST APIs.

### What This Project Covers
- Spring Boot application setup
- Creating RESTful APIs
- Handling CRUD operations
- Basic user and data management
- Connecting APIs with a database
- Understanding controller–service–repository structure

### Project Structure
- Controller layer for handling API requests
- Service layer for business logic
- Repository layer for database interaction
- Entity/model classes for data representation
- Configuration files for application setup

### Tech Stack
- Java
- Spring Boot
- REST APIs
- Database (basic integration)
- Maven

### Purpose
This project was created as part of my backend learning journey to
understand how CRM-like systems work at a basic level and how backend
services interact with databases.
